# PPE Reminder

A Flutter Android app for mining operations. Select your work area and get a voice-guided PPE checklist before starting your shift.

## Features

- Work area categories: Surface, Underground Shop, Underground Drift, Welding, Travel Home
- Voice prompts (TTS) for each checklist item
- Progress tracking with visual indicator
- Tap to check off items — fully offline, no internet required

## Setup

```bash
flutter pub get
flutter run
```

Requires Android minSdk 26+.
